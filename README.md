这个仓库用于记录推荐系统研究训练过程中的学习计划、每日日志、课程作业与知识沉淀。

目录结构

learning_plan.md：阶段性学习目标、任务拆解与进度安排。
daily_logs：按日期记录每日学习内容、实验进展和问题复盘。
assignment：课程作业、实践任务与提交记录。
knowledge_base：长期复用的知识卡片、论文阅读、工具用法与方法总结。

使用建议

每天在 daily_logs/ 中记录当天完成事项、遇到的问题和明日计划。
学到的稳定知识沉淀到 knowledge_base/，避免只停留在流水账。
作业相关内容统一放入 assignment/，保留题目、思路、代码链接和复盘。
每周回看 learning_plan.md，根据进展调整优先级。
