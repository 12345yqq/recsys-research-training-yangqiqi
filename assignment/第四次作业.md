# 推荐方向卡片：序列推荐 (Sequential Recommendation)

## 一、方向选择
- **宽方向**：序列推荐
- **细分方向**：对比学习 + 序列推荐

## 二、至少5个中英文术语
| 中文术语   | 英文术语                         |
| ------ | ---------------------------- |
| 序列推荐   | Sequential Recommendation    |
| 会话推荐   | Session-based Recommendation |
| 用户行为序列 | User Behavior Sequence       |
| 自注意力机制 | Self-Attention Mechanism     |
| 对比学习   | Contrastive Learning         |
| 数据增强   | Data Augmentation            |
| 负采样    | Negative Sampling            |

## 三、至少2类方法
1. **基于Transformer的序列建模方法**：如SASRec，利用自注意力机制捕捉用户行为序列中的长距离依赖关系。
2. **基于对比学习的序列增强方法**：如CLS4Rec，通过构造序列的正负样本来学习更鲁棒的用户表示。

## 四、至少1种任务或数据形式
- **任务类型**：下一项推荐 (Next Item Recommendation) —— 给定用户的历史交互序列，预测用户下一刻最可能交互的物品。
- **常用数据集**：Amazon Reviews、MovieLens、Yelp。

## 五、至少2个评价指标
| 指标 | 英文 | 说明 |
|------|------|------|
| 召回率 | Recall@K | 测试集中的正样本是否出现在推荐列表的前K个中 |
| 归一化折损累计增益 | NDCG@K | 衡量推荐列表中正样本的排名质量，越靠前得分越高 |
